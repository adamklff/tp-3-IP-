# TP 3 : Découpage Avancé et Analyse Binaire

## Objectifs du travail
Ce troisième travail pratique approfondit les notions de subnetting en se concentrant sur :
- Le calcul de sous-réseaux à partir d'adresses IP spécifiques.
- L'analyse binaire des masques et des adresses.
- L'identification des plages d'adresses utilisables et des passerelles (gateways).

## Technologies utilisées
- **Calcul binaire** : Conversion décimal/binaire pour l'analyse IP.
- **Subnetting** : Découpage FLSM (Fixed Length Subnet Mask).

## Contenu du dépôt
Ce dépôt contient :
1. Les solutions des exercices de découpage (/15 vers /18 et /20 vers /22).
2. Un rapport technique détaillé incluant les tableaux de correspondances binaires.

## Instructions
Pour compiler le rapport LaTeX :
